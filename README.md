# Ecommerce SQL Analysis

SQL-based ecommerce analysis covering KPIs, product contribution, and sales trends.

---

## Project Overview
This project explores ecommerce sales data using SQL to extract insights on performance, profitability and trends.

---

## Tools Used
- MySQL
- SQL (aggregations, window functions, ranking)

---

## Key Analysis
- Category and sub-category performance
- Product-level revenue and profit analysis
- Top customer analysis
- Monthly sales and profit trends
- Pareto analysis (80/20 rule)
- Product contribution within categories

---

## Key Insights
- Clear monthly trends in sales and profit
- Revenue and profit are relatively evenly distributed across categories
- No extreme product concentration observed in the dataset

## Files
- `ecommerce_analytics_queries.sql` - full MySQL analysis queries
